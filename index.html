My Projects

Project 1

Image Converter

A project that transforms a given image into a cartoon-like version. The process involves several image processing techniques, including grayscale conversion, median blur, adaptive thresholding for edge detection, and bilateral filtering for color smoothing. The result is a stylized image with simplified colors and emphasized edges, resembling a cartoon. Applications of such image processing techniques include creating artistic effects for images, developing filters for photo editing applications, and even in computer vision for pre-processing images before analysis. This specific project allows users to quickly generate cartoon-style representations of images for creative or aesthetic purposes.
